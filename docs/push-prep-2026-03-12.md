## Release Intro

Vclaw `2026.3.12` tightens the local-first AgentOS path from setup to execution. The release adds
workspace scaffolding for multi-agent prompt configuration, replayable session inspection, same-session
memory recall in task execution, and smoother chat continuity without breaking legacy `openclaw`
ecosystem compatibility.

## Main Changes

- Added `setup-workspace` to seed `AGENTS.md`, `SOUL.md`, `IDENTITY.md`, `USER.md`, and `TOOLS.md`
- Added `inspect-session` with replay turns, role traces, and continuity metadata
- Injected same-session `memoryContext` and `sessionReplay` into AgentOS task execution
- Improved chat startup with resume hints and memory-hit visibility
- Updated README, CLI docs, examples, release checklist, and release notes to match the real surface
