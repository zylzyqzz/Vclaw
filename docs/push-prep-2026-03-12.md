# Vclaw Push Prep

Version: `2026.3.13`

Status: pushed to origin/main

## Release Intro

Vclaw `2026.3.13` adds portable workspace brain packing so memory behavior and effective skills can
move with the workspace. The release also keeps the local-first AgentOS path stable, preserves
legacy `openclaw` ecosystem compatibility, and makes reinstalls much smoother.

## Main Changes

- Added `vclaw memory pack` to write a portable workspace brain manifest
- Added workspace brain fallback for memory backend and memory search recovery
- Added same-workspace merged skill sync so packed workspaces carry their usable skills
- Converted memory search loading to a backend registry for future pluggable adapters
- Updated README and versioned docs to document the reinstall-and-restore path
